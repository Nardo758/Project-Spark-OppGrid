# OppGrid Enhanced UI Implementation Plan
## Idea Card & Consultant Studio Analysis Results

**Version:** 1.0  
**Date:** April 2026  
**Target:** Replit Agent Implementation

---

## Executive Summary

This document provides implementation instructions for enhancing two critical UI components:

1. **Consultant Studio Analysis Results** — The initial free assessment for all 4 input modes (Validate Idea, Search Ideas, Identify Location, Clone Success)
2. **Idea Card (Opportunity Card)** — The full opportunity detail view with tiered layer disclosure

The goal is to replace sparse placeholder content with rich, institutionally-backed data that demonstrates research depth while maintaining clear upgrade pathways.

---

## Part 1: Consultant Studio Analysis Results

### Current State (Problem)
The Analysis Results panel shows minimal data:
- "Competitors: 0" / "Density: low"
- Generic site cards with vague labels
- No data source attribution
- No clear conversion path to paid reports

### Target State
Each of the 4 modes should display a rich but concise preview with:
- 3-4 key metrics with real numbers
- 1-2 actionable insights with specific data
- Source attribution footer
- Clear CTA tied to relevant paid report

### Implementation: Types & Interfaces

Create or update the following TypeScript interfaces:

```typescript
// frontend/src/types/analysis.ts

export interface AnalysisVerdict {
  icon: string;  // emoji: 📊, 🔥, 📍, 🎯
  label: string;  // "OppGrid verdict", "Category outlook", "Location intelligence", "Clone assessment"
  signal: 'green' | 'yellow' | 'red';
  signalText: string;  // "Worth exploring", "Proceed with targeting", "High risk", etc.
  summary: string;  // 2-3 sentence synthesis (supports HTML for <strong> emphasis)
}

export interface AnalysisMetric {
  label: string;
  value: string | number;
  subtext?: string;
  trend?: 'up' | 'down' | 'neutral';
  color?: 'success' | 'warning' | 'danger' | 'info';
}

export interface AnalysisInsight {
  type: 'positive' | 'caution' | 'info';
  label: string;
  text: string;
}

export interface SiteRecommendation {
  name: string;
  score: number; // 0-100
  scoreLabel: 'high' | 'medium' | 'low';
  description: string;
}

export interface AnalysisResult {
  mode: 'validate_idea' | 'search_ideas' | 'identify_location' | 'clone_success';
  verdict: AnalysisVerdict;  // THE KEY ADDITION - synthesis paragraph
  metrics: AnalysisMetric[];
  insights: AnalysisInsight[];
  sites?: SiteRecommendation[]; // For Identify Location mode
  tags?: string[]; // Data source tags
  cta: {
    text: string;
    reportType: string;
    price: number;
  };
  lastUpdated: string;
  processingTime: number; // in seconds
}
```

### Implementation: API Endpoint

Create or update the analysis endpoint to return richer data:

```python
# backend/app/api/routes/consultant_studio.py

from fastapi import APIRouter, Depends
from app.services.analysis_service import AnalysisService

router = APIRouter()

@router.post("/analyze")
async def analyze(
    request: AnalysisRequest,
    user_id: int = Depends(get_current_user_id)  # FIX: Remove hardcoded user_id = 1
) -> AnalysisResult:
    """
    Returns rich analysis results for Consultant Studio.
    
    Mode-specific returns:
    - validate_idea: validation_score, market_signals, competition, demand_insight, risk_insight
    - search_ideas: opportunity_count, trending_pct, avg_score, top_signals
    - identify_location: competitors, density, avg_rating, foot_traffic, demographics, sites
    - clone_success: viability, startup_cost, market_gap, why_works, differentiation
    """
    service = AnalysisService()
    return await service.analyze(request, user_id)
```

### Implementation: Analysis Service

```python
# backend/app/services/analysis_service.py

class AnalysisService:
    
    async def analyze_location(self, location: str, business_type: str) -> dict:
        """
        Identify Location mode analysis.
        
        Data sources to query:
        1. Google Places API → competitor count, avg rating
        2. Census Bureau API → demographics (median income, age bands, pop growth)
        3. Walk Score API → walkability score
        4. Internal PostGIS → geographic density calculation
        5. Foot traffic estimates (derived from Google Popular Times if available)
        """
        
        # Query competitor data
        competitors = await self.google_places.search_nearby(
            location=location,
            business_type=business_type,
            radius_miles=5
        )
        
        # Calculate density ratio
        population = await self.census.get_population(location)
        density_ratio = len(competitors) / (population / 1000) if population else 0
        density_label = self._calculate_density_label(density_ratio, business_type)
        
        # Get demographics
        demographics = await self.census.get_demographics(location)
        
        # Score micro-markets
        sites = await self._score_micro_markets(location, business_type, competitors)
        
        # Generate verdict using Claude
        verdict = await self._generate_verdict(
            mode="identify_location",
            data={
                "location": location,
                "business_type": business_type,
                "competitor_count": len(competitors),
                "density_label": density_label,
                "density_ratio": f"1 per {int(population / len(competitors))} residents" if competitors else "N/A",
                "foot_traffic_yoy": demographics.get('foot_traffic_yoy', 0),
                "age_25_44_pct": demographics.get('age_25_44_pct', 0),
                "top_sites": [s['name'] for s in sites[:2]],
                "avoid_site": sites[-1]['name'] if len(sites) > 2 else None
            }
        )
        
        return {
            "mode": "identify_location",
            "verdict": verdict,
            "metrics": [
                {
                    "label": "Competitors",
                    "value": len(competitors),
                    "subtext": "within 5 mi"
                },
                {
                    "label": "Density",
                    "value": density_label,
                    "subtext": f"1 per {int(population / len(competitors))} residents" if competitors else "N/A",
                    "color": "danger" if density_label == "High" else "warning" if density_label == "Medium" else "success"
                },
                {
                    "label": "Avg. rating",
                    "value": f"{self._avg_rating(competitors):.1f}★",
                    "subtext": "area benchmark"
                },
                {
                    "label": "Foot traffic",
                    "value": f"+{demographics.get('foot_traffic_yoy', 0)}%",
                    "subtext": "YoY growth",
                    "trend": "up" if demographics.get('foot_traffic_yoy', 0) > 0 else "down"
                }
            ],
            "demographics": {
                "median_income": demographics.get("median_income"),
                "age_25_44_pct": demographics.get("age_25_44_pct"),
                "pop_growth": demographics.get("pop_growth_pct"),
                "walk_score": demographics.get("walk_score")
            },
            "sites": sites[:3],  # Top 3 micro-markets
            "tags": ["Census Bureau", "Google Places", "Walk Score", "PostGIS analytics"],
            "cta": {
                "text": "Full location analysis with Census data, heat maps & lease intel",
                "reportType": "Business Plan",
                "price": 149
            }
        }
    
    async def analyze_idea(self, idea: str) -> dict:
        """
        Validate Idea mode analysis.
        
        Data sources:
        1. Internal opportunity database (DeepSeek search)
        2. Reddit API (related discussions)
        3. Google Trends (search interest)
        4. Competitive analysis (existing players)
        """
        
        # Search internal opportunities
        related_opportunities = await self.deepseek.search_opportunities(idea)
        
        # Calculate validation score
        validation_score = self._calculate_validation_score(related_opportunities)
        
        # Get market signals
        signals = await self.scraper_aggregator.get_signal_count(idea, days=90)
        
        # Analyze competition
        competition = await self._analyze_competition(idea)
        
        # Extract key demand signal
        demand_signal = self._extract_top_demand_signal(related_opportunities)
        
        return {
            "mode": "validate_idea",
            "metrics": [
                {
                    "label": "Validation score",
                    "value": f"{validation_score}/100",
                    "subtext": self._score_label(validation_score),
                    "color": "success" if validation_score >= 70 else "warning" if validation_score >= 50 else "danger"
                },
                {
                    "label": "Market signals",
                    "value": signals["count"],
                    "subtext": "Last 90 days"
                },
                {
                    "label": "Competition",
                    "value": competition["level"],
                    "subtext": f"{competition['count']} active players",
                    "color": "success" if competition["level"] == "Low" else "warning"
                }
            ],
            "insights": [
                {
                    "type": "positive",
                    "label": "Key demand signal",
                    "text": demand_signal
                },
                {
                    "type": "caution",
                    "label": "Market risk",
                    "text": competition["risk_summary"]
                }
            ],
            "tags": ["Reddit r/entrepreneur", "Google Trends", "Crunchbase", "+4 sources"],
            "cta": {
                "text": "Full competitive analysis, TAM/SAM, and execution playbook",
                "reportType": "Feasibility Study",
                "price": 25
            }
        }
    
    async def _generate_verdict(self, mode: str, data: dict) -> dict:
        """
        Generate the OppGrid Verdict synthesis using Claude.
        
        The verdict should:
        1. Reference specific metrics from the analysis
        2. Give a clear recommendation (proceed, proceed with caution, pivot, etc.)
        3. Highlight the key insight or differentiator
        4. Be 2-3 sentences max
        """
        
        prompts = {
            "validate_idea": f"""
                Based on this analysis:
                - Validation score: {data['score']}/100
                - Market signals: {data['signal_count']} in last 90 days
                - Competition: {data['competition_level']} ({data['competitor_count']} players)
                - Key demand signal: {data['demand_signal']}
                - Risk: {data['risk_summary']}
                
                Write a 2-3 sentence verdict. Reference the specific metrics. 
                If score >= 70, recommend proceeding. If 50-69, proceed with caution.
                Always highlight the key opportunity or differentiation angle.
                Use <strong> tags for emphasis on 1-2 key phrases.
            """,
            
            "identify_location": f"""
                Based on this location analysis for {data['business_type']} in {data['location']}:
                - Competitors: {data['competitor_count']}
                - Market density: {data['density_label']} ({data['density_ratio']})
                - Foot traffic trend: +{data['foot_traffic_yoy']}% YoY
                - Target demographic (25-44): {data['age_25_44_pct']}%
                - Best micro-markets: {', '.join(data['top_sites'])}
                - Caution area: {data['avoid_site']}
                
                Write a 2-3 sentence verdict. Be specific about WHERE to open.
                If high density, recommend specific neighborhoods with lower saturation.
                Use <strong> tags for emphasis on recommended areas.
            """,
            
            "clone_success": f"""
                Based on this clone analysis for {data['model_name']}:
                - Model viability: {data['viability']}
                - Estimated startup cost: ${data['startup_cost']}
                - Market gap: {data['market_gap']}% underserved metros
                - Success factors: {', '.join(data['success_factors'])}
                - Differentiation needed: {data['differentiation']}
                
                Write a 2-3 sentence verdict. Reference why this model works.
                Highlight the lean/modified approach vs. the original format.
                Use <strong> tags for emphasis on key numbers.
            """,
            
            "search_ideas": f"""
                Based on this category analysis for {data['category']}:
                - Total opportunities: {data['opportunity_count']}
                - Trending: {data['trending_pct']}% vs last month
                - Average viability score: {data['avg_score']}
                - Top signals: {', '.join(data['top_signals'])}
                
                Write a 2-3 sentence verdict about the category outlook.
                Reference the trending signals and overall category health.
                Use <strong> tags for emphasis.
            """
        }
        
        # Call Claude to generate the verdict
        response = await self.claude.generate(
            system="You are an expert market analyst. Write concise, data-driven verdicts.",
            prompt=prompts[mode]
        )
        
        # Determine signal color based on mode-specific logic
        signal = self._determine_signal_color(mode, data)
        
        return {
            "icon": self._get_verdict_icon(mode),
            "label": self._get_verdict_label(mode),
            "signal": signal["color"],
            "signalText": signal["text"],
            "summary": response.text
        }
    
    def _get_verdict_icon(self, mode: str) -> str:
        return {
            "validate_idea": "📊",
            "search_ideas": "🔥",
            "identify_location": "📍",
            "clone_success": "🎯"
        }.get(mode, "📊")
    
    def _get_verdict_label(self, mode: str) -> str:
        return {
            "validate_idea": "OppGrid verdict",
            "search_ideas": "Category outlook",
            "identify_location": "Location intelligence",
            "clone_success": "Clone assessment"
        }.get(mode, "Analysis")
    
    def _determine_signal_color(self, mode: str, data: dict) -> dict:
        """Determine green/yellow/red signal based on mode-specific thresholds."""
        
        if mode == "validate_idea":
            score = data.get("score", 0)
            if score >= 70:
                return {"color": "green", "text": "Worth exploring"}
            elif score >= 50:
                return {"color": "yellow", "text": "Proceed with caution"}
            else:
                return {"color": "red", "text": "High risk"}
        
        elif mode == "identify_location":
            density = data.get("density_label", "")
            if density == "Low":
                return {"color": "green", "text": "Strong opportunity"}
            elif density == "Medium":
                return {"color": "green", "text": "Proceed with targeting"}
            else:
                return {"color": "yellow", "text": "Proceed with targeting"}
        
        elif mode == "clone_success":
            viability = data.get("viability", "")
            if viability == "High":
                return {"color": "green", "text": "High replicability"}
            elif viability == "Medium":
                return {"color": "yellow", "text": "Moderate replicability"}
            else:
                return {"color": "red", "text": "Low replicability"}
        
        elif mode == "search_ideas":
            trending = data.get("trending_pct", 0)
            if trending > 10:
                return {"color": "green", "text": "High activity"}
            elif trending > 0:
                return {"color": "yellow", "text": "Stable"}
            else:
                return {"color": "red", "text": "Declining"}
        
        return {"color": "yellow", "text": "Review data"}
```

### Implementation: React Component

```tsx
// frontend/src/components/ConsultantStudio/AnalysisResults.tsx

import React from 'react';
import { AnalysisResult, AnalysisMetric, AnalysisInsight, SiteRecommendation } from '@/types/analysis';

interface AnalysisResultsProps {
  result: AnalysisResult | null;
  isLoading: boolean;
}

export const AnalysisResults: React.FC<AnalysisResultsProps> = ({ result, isLoading }) => {
  if (isLoading) {
    return <AnalysisResultsSkeleton />;
  }
  
  if (!result) {
    return null;
  }
  
  return (
    <div className="analysis-results">
      <div className="results-header">
        <div className="results-title">
          <span className="indicator" />
          <span>Analysis Results</span>
        </div>
        <button className="close-btn" aria-label="Close">×</button>
      </div>
      
      {/* VERDICT BOX - The key synthesis paragraph */}
      <VerdictBox verdict={result.verdict} />
      
      {/* Metrics Grid */}
      <div className={`metrics-grid metrics-${result.metrics.length}`}>
        {result.metrics.map((metric, idx) => (
          <MetricCard key={idx} metric={metric} />
        ))}
      </div>
      
      {/* Demographics (Location mode only) */}
      {result.mode === 'identify_location' && result.demographics && (
        <DemographicsSnapshot demographics={result.demographics} />
      )}
      
      {/* Insights */}
      {result.insights && result.insights.length > 0 && (
        <div className="insights-section">
          {result.insights.map((insight, idx) => (
            <InsightRow key={idx} insight={insight} />
          ))}
        </div>
      )}
      
      {/* Site Recommendations (Location mode only) */}
      {result.sites && result.sites.length > 0 && (
        <div className="sites-section">
          <div className="section-label">Recommended micro-markets</div>
          {result.sites.map((site, idx) => (
            <SiteCard key={idx} site={site} />
          ))}
        </div>
      )}
      
      {/* Source Tags */}
      {result.tags && (
        <div className="source-tags">
          {result.tags.map((tag, idx) => (
            <span key={idx} className="tag">{tag}</span>
          ))}
        </div>
      )}
      
      {/* CTA */}
      <div className="cta-row">
        <span className="cta-text">{result.cta.text}</span>
        <button className="cta-btn">
          Get {result.cta.reportType} → ${result.cta.price}
        </button>
      </div>
      
      {/* Footer */}
      <div className="results-footer">
        Powered by DeepSeek + Claude AI · {result.processingTime}s
      </div>
    </div>
  );
};

// Sub-components
const MetricCard: React.FC<{ metric: AnalysisMetric }> = ({ metric }) => (
  <div className="metric-card">
    <div className="metric-label">{metric.label}</div>
    <div className={`metric-value ${metric.color || ''}`}>
      {metric.trend === 'up' && <span className="trend-arrow">↑</span>}
      {metric.trend === 'down' && <span className="trend-arrow">↓</span>}
      {metric.value}
    </div>
    {metric.subtext && <div className="metric-sub">{metric.subtext}</div>}
  </div>
);

const InsightRow: React.FC<{ insight: AnalysisInsight }> = ({ insight }) => (

      <span className="insight-label">{insight.label}</span>
```

```python
@router.get("/{opportunity_id}")
async def get_opportunity(
    opportunity_id: str,
    user_id: int = Depends(get_current_user_id)  # FIX: Remove hardcoded user_id = 1
) -> OpportunityCard:
    """
    Returns full opportunity card with layer-based access control.
    
    Layer 1 (FREE/$15): Problem statement, problem detail, basic validation
    Layer 2 ($99+): Full sources, competitors, differentiation insights
    Layer 3 ($499+): Solution pathways, execution playbook, 90-day timeline
    """
    
    opportunity = await opportunity_service.get(opportunity_id)
    user_tier = await subscription_service.get_tier(user_id)
    
    # Determine layer access
    layer_access = calculate_layer_access(user_tier, opportunity.age_in_days)
    
    # Build response with gating
    return build_opportunity_response(opportunity, layer_access)


def build_opportunity_response(opportunity, layer_access: int) -> dict:
    """
    Build response with appropriate gating.
    
    - Sources: Show top 3, gate rest if layer < 2
    - Competitors: Show top 2, gate rest if layer < 2
    - Solution pathways: Fully gated if layer < 3
    """
    
    sources = opportunity.sources[:3]  # Always show top 3
    if layer_access >= 2:
        sources = opportunity.sources  # Show all
    else:
        # Add gated placeholders
        for i in range(3, min(6, opportunity.sources_total)):
            sources.append({
                "isGated": True,
                "platform": "unknown",
                "title": f"+{opportunity.sources_total - 3} more sources"
            })
    
    # Similar logic for competitors
    competitors = opportunity.competitors[:2]
    if layer_access < 2:
        competitors.append({
            "isGated": True,
            "name": f"+{opportunity.competitors_total - 2} more competitors",
            "description": "Full SWOT analysis available"
        })
    
    return {
        # ... full response structure
        "researchDashboard": {
            "currentLayer": layer_access,
            "sources": sources,
            "sourcesTotal": opportunity.sources_total,
            "competitors": competitors,
            "competitorsTotal": opportunity.competitors_total,
            # ...
        }
    }
```
