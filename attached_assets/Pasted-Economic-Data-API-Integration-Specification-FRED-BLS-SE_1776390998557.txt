  Date               April 2026

  Priority           HIGH - Prerequisite for credible financial projections

Executive Summary

This specification adds three external data APIs to OppGrid's report pipeline: FRED (macroeconomic indicators), BLS (industry labor data), and SEC-API.io (public company financials). Together they provide the economic context, labor benchmarks, and industry operating metrics needed to ground financial projections in verifiable data.

+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| WHY THIS MATTERS                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Current reports cite statistics with no source. "The self-storage industry averages 72% operating margins" — where did that come from? With these APIs wired to the pipeline, Claude receives actual data: PSA's 10-K shows 74.2% NOI margin, BLS QCEW shows industry employment up 3.1% YoY, FRED shows fed funds at 4.25%. Reports cite "SEC 10-K FY2025" and "BLS QCEW 2025 Q4" inline. This transforms perceived quality from "AI-generated" to "data-driven." |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

1. FRED API Integration

1.1 API Overview

Base URL: https://api.stlouisfed.org/fred/

Authentication: API key (free registration at fred.stlouisfed.org)

Rate Limit: 120 requests per minute

Cost: Free

1.2 Key Series for OppGrid Reports

  -------------- ---------------------------------- ---------------- ------------------------
  Series ID      Description                        Frequency        Report Section

  GDP            Real Gross Domestic Product        Quarterly        Economic Context

  CPIAUCSL       Consumer Price Index (All Urban)   Monthly          Pricing Strategy

  FEDFUNDS       Federal Funds Effective Rate       Daily            Financing Costs

  UMCSENT        U. Michigan Consumer Sentiment     Monthly          Market Timing

  UNRATE         Unemployment Rate                  Monthly          Labor Market

  MORTGAGE30US   30-Year Fixed Mortgage Rate        Weekly           Financing, Real Estate
  -------------- ---------------------------------- ---------------- ------------------------

3. SEC-API.io Integration

3.1 API Overview

Base URL: https://api.sec-api.io/

Authentication: API key in Authorization header

Rate Limit: 100 requests/minute on Starter plan

Cost: $49/month (Starter plan)

+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| WHY SEC-API.io INSTEAD OF RAW EDGAR                                                                                                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                          |
| Raw SEC EDGAR filings are XBRL (structured) + HTML (messy). Different companies tag the same metrics differently — "Revenue" vs "Total Revenues" vs "Net Sales". SEC-API.io pre-parses everything into normalized JSON, saving weeks of parsing dev work. You get clean income statements, balance sheets, and key metrics ready to use. |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

3.3 What You Extract from 10-Ks

For a "Self Storage in Miami" report, pulling from PSA, EXR, CUBE gives you:

-   Revenue per square foot: $15-22/sq ft (varies by market)

-   Stabilized occupancy: 92-95%

-   Operating margins: 70-75% NOI margin

-   Same-store revenue growth: 5-8% annually

-   Customer acquisition cost: $50-150 per move-in

-   Average tenant duration: 14-18 months

This is exactly what grounds a financial model in reality instead of Claude making up numbers.

|                                                                          |
| class SECAPIService:                                                     |
|                                                                          |
| @staticmethod                                                            |
|                                                                          |
| def get_comps_for_business(business_type: str) -> List[str]:             |
|                                                                          |
| """Map business type to relevant public company tickers."""              |
|                                                                          |
| business_lower = business_type.lower()                                   |
|                                                                          |
| for key, tickers in INDUSTRY_COMPS.items():                              |
|                                                                          |
| if key in business_lower:                                                |
|                                                                          |
| return tickers                                                           |
|                                                                          |
| return []                                                                |
|                                                                          |
| @staticmethod                                                            |
|                                                                          |
| async def get_latest_10k(ticker: str) -> Optional[Dict]:                 |
|                                                                          |
| """Fetch most recent 10-K filing for a ticker."""                        |
|                                                                          |
| query = {                                                                |
|                                                                          |
| "query": {"query_string": {                                              |
|                                                                          |
| "query": f'ticker:{ticker} AND formType:"10-K"'                          |
|                                                                          |
| }},                                                                      |
|                                                                          |
| "from": 0, "size": 1,                                                    |
|                                                                          |
| "sort": [{"filedAt": {"order": "desc"}}]                                 |
|                                                                          |
| }                                                                        |
|                                                                          |
| async with httpx.AsyncClient() as client:                                |
|                                                                          |
| resp = await client.post(                                                |
|                                                                          |
| SEC_API_BASE,                                                            |
|                                                                          |
| json=query,                                                              |
|                                                                          |
| headers={'Authorization': SEC_API_KEY},                                  |
|                                                                          |
| timeout=15.0                                                             |
|                                                                          |
| )                                                                        |
|                                                                          |
| resp.raise_for_status()                                                  |
|                                                                          |
| data = resp.json()                                                       |
|                                                                          |
| if data.get('filings'):                                                  |
|                                                                          |
| return data['filings'][0]                                                |
|                                                                          |
| return None                                                              |

+-----------------------------------------------------------------------+
| @staticmethod                                                         |
|                                                                       |
| async def get_financials(accession_no: str) -> Optional[Dict]:        |
|                                                                       |
| """Get parsed financial statements from XBRL."""                      |
|                                                                       |
| async with httpx.AsyncClient() as client:                             |
|                                                                       |
| resp = await client.get(                                              |
|                                                                       |
| f'{SEC_API_BASE}/xbrl-to-json',                                       |
|                                                                       |
| params={'accession-no': accession_no, 'token': SEC_API_KEY},          |
|                                                                       |
| timeout=15.0                                                          |
|                                                                       |
| )                                                                     |
|                                                                       |
| resp.raise_for_status()                                               |
|                                                                       |
| return resp.json()                                                    |
|                                                                       |
| @staticmethod                                                         |
|                                                                       |
| async def get_industry_benchmarks(                                    |
|                                                                       |
| business_type: str                                                    |
|                                                                       |
| ) -> Optional[IndustryBenchmarks]:                                    |
|                                                                       |
| """                                                                   |
|                                                                       |
| Fetch financials for all relevant comps,                              |
|                                                                       |
| calculate industry averages.                                          |
|                                                                       |
| """                                                                   |
|                                                                       |
| tickers = SECAPIService.get_comps_for_business(business_type)         |
|                                                                       |
| if not tickers:                                                       |
|                                                                       |
| return None                                                           |
|                                                                       |
| comps = []                                                            |
|                                                                       |
| for ticker in tickers[:3]: # Limit to top 3 comps                     |
|                                                                       |
| filing = await SECAPIService.get_latest_10k(ticker)                   |
|                                                                       |
| if filing:                                                            |
|                                                                       |
| financials = await SECAPIService.get_financials(                      |
|                                                                       |
| filing['accessionNo']                                                 |
|                                                                       |
| )                                                                     |
|                                                                       |
| if financials:                                                        |
|                                                                       |
| comps.append(PublicCompData(                                          |
|                                                                       |
| ticker=ticker,                                                        |
|                                                                       |
| company_name=filing.get('companyName'),                               |
|                                                                       |
| fiscal_year=filing.get('fiscalYear'),                                 |
|                                                                       |
| revenue=financials.get('Revenues'),                                   |
|                                                                       |
| operating_income=financials.get('OperatingIncome'),                   |
|                                                                       |
| source=f'SEC 10-K FY{filing.get("fiscalYear")}'                       |
|                                                                       |
| ))                                                                    |
|                                                                       |
| # Calculate averages                                                  |
|                                                                       |
| if comps:                                                             |
|                                                                       |
| avg_margin = sum(c.operating_margin for c in comps) / len(comps)      |
|                                                                       |
| return IndustryBenchmarks(                                            |
|                                                                       |
| avg_operating_margin=avg_margin,                                      |
|                                                                       |
| public_comps=comps,                                                   |
|                                                                       |
| source='SEC 10-K filings via sec-api.io'                              |
|                                                                       |
| )                                                                     |
|                                                                       |
| return None                                                           |
